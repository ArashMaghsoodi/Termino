# **App Name**: Termino

## Core Features:

- User Authentication: Allows users to create accounts, log in, and manage their profiles.
- Course Catalog: Displays a searchable and filterable list of available courses with details such as schedule, instructor, and prerequisites.
- Schedule Builder: Enables students to create potential schedules by selecting courses and viewing them on a weekly calendar.
- Conflict Detection: Identifies time conflicts between selected courses in the schedule.
- Theme Toggle: Switch between light and dark themes for comfortable viewing.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) for a professional and academic feel.
- Background color: Light Gray (#F5F5F5) for light mode, Dark Gray (#303030) for dark mode.
- Accent color: Amber (#FFC107) for highlights and interactive elements.
- Body and headline font: 'Vazir', a modern and readable Persian sans-serif font.
- Use simple and consistent icons for navigation and course information.
- Right-to-left layout with a fixed navbar at the top and a centered main content area.
- Smooth transitions and hover effects for a modern and responsive user experience.
